# ❤️ For Aku

A cinematic Apple-inspired apology website built with HTML, CSS and JavaScript.

## ✨ Features

- 🌌 Animated starry night
- ✨ Glowing particles
- 🎹 Background piano music
- 📱 Mobile swipe navigation
- 💎 Glassmorphism UI
- 🎬 Cinematic transitions
- ❤️ Interactive ending

## 📂 Project Structure

```
for-akku/
│
├── index.html
├── style.css
├── script.js
│
└── assets/
    ├── piano.mp3
    ├── favicon.png
    ├── poster.jpg
```

## 🚀 Deploy on GitHub Pages

1. Open your repository.
2. Go to **Settings → Pages**.
3. Under **Source**, choose:
   - Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
4. Save.
5. Wait a minute for GitHub to publish your site.
